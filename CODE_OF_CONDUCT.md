# Code Of Conduct

## Our Pledge

We are committed to making participation in this project respectful,
professional, and harassment-free.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experience levels.
- Keep discussions focused on defensive, authorized security work.
- Avoid sharing sensitive data, credentials, or private target information.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Requests for exploit automation, credential attacks, evasion, or destructive
  testing.
- Publishing private program data or target evidence without authorization.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributions that
violate this code of conduct.
