# Security Policy

## Legal And Ethical Use

bb-assistant is designed for authorized, defensive bug bounty work only. Use it
only on assets where you have explicit permission from the program owner and
where the target is in scope.

## Capability Boundaries

This project intentionally does not include:

- Exploit automation
- Bruteforce
- Credential stuffing
- Password spraying
- DoS or DDoS behavior
- Social engineering
- WAF bypass
- Stealth or evasion logic
- Data exfiltration
- Active vulnerability exploitation

## Reporting Security Issues In This Project

If you find a security issue in bb-assistant itself, open a private advisory or
contact the maintainers through the repository's preferred contact channel.
Please include:

- A concise description of the issue
- Steps to reproduce
- Affected version or commit
- Suggested remediation, if known

Do not include real third-party secrets, private program data, or evidence from
targets where you do not have authorization.
